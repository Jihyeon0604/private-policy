# private-policy

개인정보처리방침

Sketch Catch는 사용자의 개인정보를 소중히 여기며, 다음과 같은 방식으로 수집 및 이용합니다.

1. 수집하는 정보
- 사용자가 업로드한 이미지
- 앱 사용 시 생성되는 로그 정보 (에러 로그, 시간 등)
- 사용자의 기기 정보 (기종, OS 버전 등)

2. 정보의 이용 목적
- 이미지 인식 결과 제공
- 서비스 품질 개선 및 버그 수정

3. 제3자 제공
- 어떠한 개인정보도 외부에 제공하지 않습니다.

4. 보관 및 삭제
- 서버에 저장된 이미지는 분석 완료 후 즉시 삭제됩니다.
- 기타 로그는 최대 30일간 보관 후 자동 삭제됩니다.
